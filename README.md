# 📊 Sales Dashboard (Power BI)

## 🚀 Overview
An interactive Power BI dashboard analyzing sales, profit, and customer trends using Superstore dataset.

## 📊 Features
- Sales & Profit KPIs
- Category & Region analysis
- Time-based trends
- Interactive filters
